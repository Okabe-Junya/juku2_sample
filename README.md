# Gopher塾 #2 サンプルリポジトリ

## 概要

[Gopher塾 #2](https://tenntenn.connpass.com/event/268056/) に関する記事のサンプルコードです．`ls` コマンドをGoらしく書き換えてみたものです．

## 使い方

### インストール

```txt
% git clone https://github.com/Okabe-Junya/juku2_sample.git
```

### コンパイル

```txt
% cd juku2_sample
% make
```

### 実行

```txt
% ./bin/myls
```
